# 🌟 마음일기

Firebase 기반 캘린더 일기장 웹 서비스

## 🚀 기능

- 📝 캘린더 기반 일기 작성
- 😊 감정 기록 및 통계
- 🤖 AI 텍스트 확장 (GPT-4o)
- 📷 이미지 업로드 (Firebase Storage)
- 🔔 푸시 알림
- 🌓 다크모드/라이트모드
- 📱 반응형 디자인

## 🛠 기술 스택

### Frontend
- React 19.1.0 + Vite
- Tailwind CSS
- Lucide React Icons

### Backend
- Firebase Authentication (카카오 로그인)
- Firestore Database
- Firebase Storage
- Firebase Hosting

### AI
- OpenAI GPT-4o API

## 🌐 배포

- **웹사이트**: https://maumilgi-1a4cb.web.app

## 📱 로컬 실행

```bash
# 프론트엔드 실행
cd frontend
npm install
npm run dev

# 프로덕션 빌드
npm run build

# Firebase 배포
firebase deploy
```
